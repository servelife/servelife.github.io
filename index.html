Welcome to ServeLife's Outpost @ GitHub.

ServeLife's mission isn't to change the world. Instead we endeavor to organize those who will. Key to this mission is equipping people with knowledge and expertise that empowers and enables. Learning projects are one such way we accomplish this. A ServeLife Learning Project is an experience that helps users transform knowledge into relevant skills to better themselves and those around them.

Open Sourced & Community-Owned

All projects start with a single path defined; guiding participants through a series of recipes (or set of instructions) that incrementally accomplishes the project's goals. Yet ServeLife Learning Projects are open sourced and community-owned. Anyone can create and contribue new learning projects and alternative recipes. It's easy to do. Fork a project or recipe, make your changes and submit a pull request.

Learning Projects

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator written by our own Tom Preston-Werner. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
